# Rogers Chromecast Receiver

## Manage Release
1. Select "Create a new release" on the homepage of your repo
2. Enter your release version as a Title and a Tag: `1.0.0`
3. Publish the release
4. Now go to Deploy action
5. Click on "Run workflow" and select your branch and environment where it should be released 
6. Enter your release version under `Release Tag`
7. Run workflow
